# 人生 K 线 · 理性趋势分析工具

这是一个微信小程序 + 可企业化 SDK 的趋势分析系统，
用于可视化人生阶段趋势与风险结构。

## 功能
- 人生 K 线图
- 阶段趋势解读
- Explain Engine 可解释文本
- 普通 / 全屏模式
- 付费分层（Free / Plus / Pro）


人生K线图
## 小程序功能介绍（审核专用）
- 人生K线图 是一款以个人基础信息为输入，通过本地数据算法与趋势分析方法，对人生阶段性变化进行可视化分析与参考建议的工具型小程序。

## 本小程序主要功能包括：
- 人生阶段趋势可视化
  - 通过曲线和区间形式，展示不同年龄阶段的变化趋势
  - 类似于金融领域中的“K线/趋势图”表达方式，仅用于趋势参考
- 多维因素综合分析
  - 基于时间、个人输入信息及宏观环境变化等因素
  - 进行综合分析与结构化展示
- 理性参考建议
  - 提供对学习、工作、生活规划等方面的一般性建议
  - 所有建议均为参考性质，不构成任何承诺或结论
- 交互式查看
  - 支持年份滑动、区间查看、点击节点查看解释说明
  - 仅用于信息展示与辅助理解
