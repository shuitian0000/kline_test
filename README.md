# 人生 K 线 · 理性趋势分析工具

这是一个微信小程序 + 可企业化 SDK 的趋势分析系统，
用于可视化人生阶段趋势与风险结构。

## 功能
- 人生 K 线图
- 阶段趋势解读
- Explain Engine 可解释文本
- 普通 / 全屏模式
- 付费分层（Free / Plus / Pro）


## 人生K线图
## 副标题：理性分析 · 人生趋势参考工具

## 小程序功能介绍（审核专用）
- 人生K线图 是一款以个人基础信息为输入，通过本地数据算法与趋势分析方法，对人生阶段性变化进行可视化分析与参考建议的工具型小程序。

## 本小程序主要功能包括：
- 人生阶段趋势可视化
  - 通过曲线和区间形式，展示不同年龄阶段的变化趋势
  - 类似于金融领域中的“K线/趋势图”表达方式，仅用于趋势参考
- 多维因素综合分析
  - 基于时间、个人输入信息及宏观环境变化等因素
  - 进行综合分析与结构化展示
- 理性参考建议
  - 提供对学习、工作、生活规划等方面的一般性建议
  - 所有建议均为参考性质，不构成任何承诺或结论
- 交互式查看
  - 支持年份滑动、区间查看、点击节点查看解释说明
  - 仅用于信息展示与辅助理解


## 合规与免责声明说明（非常关键，建议完整保留）
- 本小程序为信息分析与可视化工具，不涉及任何形式的算命、占卜、预测未来结果或保证性结论
- 所有展示内容仅供用户个人参考与理性分析
- 不提供任何医疗、金融、法律、投资等专业意见
- 不涉及收费预测、付费改运、运势保证等内容
- 用户应基于自身判断与实际情况独立决策

## 使用人群说明
- 适合希望通过数据化方式，观察人生不同阶段变化趋势，并进行自我规划参考的普通用户。
